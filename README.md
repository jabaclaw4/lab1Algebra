# Лабораторная работа 1: метод Гаусса и LU-разложение

Сравнение методов решения систем линейных алгебраических уравнений (СЛАУ).

## Реализованные методы

- Метод Гаусса без выбора ведущего элемента
- Метод Гаусса с частичным выбором ведущего элемента по столбцу
- LU-разложение (алгоритм Дулиттла) + прямая и обратная подстановки

## Структура проекта
```
lab1Algebra/
├── headers/
│   ├── gauss.h        — объявления функций метода Гаусса
│   ├── lu.h           — объявления функций LU-разложения
│   ├── matrix.h       — объявления функций генерации матриц
│   └── utils.h        — объявления таймера, невязки, погрешности
├── src/
│   ├── gauss.cpp      — методы Гаусса
│   ├── lu.cpp         — LU-разложение и подстановки
│   ├── matrix.cpp     — генерация матриц (случайная, Гильберта)
│   └── utils.cpp      — таймер, невязка, относительная погрешность
├── main.cpp           — точка входа, три эксперимента
├── CMakeLists.txt
└── README.md
```

## Сборка и запуск

Открыть проект в CLion и нажать **Run**, либо собрать вручную:

```bash
mkdir build && cd build
cmake ..
cmake --build .
./lab1Algebra
```

Требуется поддержка C++17.
