# Лабораторная работа 1: метод Гаусса и LU-разложение

Сравнение методов решения систем линейных алгебраических уравнений (СЛАУ).

## Реализованные методы

- Метод Гаусса без выбора ведущего элемента
- Метод Гаусса с частичным выбором ведущего элемента по столбцу
- LU-разложение (алгоритм Дулиттла) + прямая и обратная подстановки

## Структура проекта

```
main.cpp       — точка входа, три эксперимента
gauss.cpp/h    — методы Гаусса
lu.cpp/h       — LU-разложение и подстановки
matrix.cpp/h   — генерация матриц (случайная, Гильберта)
utils.cpp/h    — таймер, невязка, относительная погрешность
CMakeLists.txt
```

## Сборка и запуск

Открыть проект в CLion и нажать **Run**, либо собрать вручную:

```bash
mkdir build && cd build
cmake ..
cmake --build .
./lab1Algebra
```

Требуется поддержка C++17.
